## 비즈니스 트랙 채점 기준

> ⚠️ 웹사이트 vs 공식 PDF에 표기 차이 있음 — 공통 핵심 축 기준 정리

### 버전 A: 웹사이트 (100점)

| 항목 | 배점 |
|------|------|
| 문제 정의 및 해결 아이디어 | 25점 |
| 데이터 분석 | 25점 |
| **비즈니스 임팩트** | **30점** (최고!) |
| 창의성 및 혁신성 | 20점 |

### 버전 B: 공식 PDF

| 항목 | 비중 |
|------|------|
| Creativity | 25% |
| Snowflake alignment | 25% |
| Marketability | 25% |
| Feasibility | 25% |
| Presentation | +10% |

### 보너스 포인트
`Snowpark` `Snowflake Intelligence` `Marketplace` `Streamlit` `Cortex AI` `Native Apps` `MCP` 활용 시 가산점

## 고득점 공식

```
좋은 점수 = 명확한 문제 정의
           + 여러 데이터셋 창의적 조합
           + "실제로 돈이 되는" 비즈니스 모델
           + 구체적 실행 로드맵
           + 설득력 있는 10분 영상
```

## ✅ 최종 체크리스트

- [ ] 문제 정의가 명확한가?
- [ ] 제공 데이터셋을 2개 이상 결합했는가?
- [ ] Snowflake Marketplace를 활용했는가?
- [ ] 비즈니스 모델이 구체적인가?
- [ ] 시장 규모/기회를 제시했는가?
- [ ] 실행 로드맵이 현실적인가?
- [ ] 10분 영상 스토리라인이 매끄러운가?
- [ ] Snowflake 기능 활용을 언급했는가?
