## 전원 소구 가능한 아이디어 3유형

### Type A: 라이프스타일 의사결정 플랫폼
> "이사할 때 최적의 동네를 추천해주는 AI"
- 부동산(김재구) + 상권(김선경) + 통신(전병기) + 이사(이하석) + 추천(김대식) + Marketplace(최기영)

### Type B: 소상공인/창업 지원
> "창업 위치 + 마케팅까지 AI가 제안하는 올인원"
- 통합(최기영) + 임대료(김재구) + 상권(김선경) + 마케팅(이하석) + 유동인구(전병기) + 트렌드(김대식)

### Type C: AI 데이터 분석 민주화
> "자연어로 질문하면 마켓플레이스 데이터를 분석해주는 AI"
- 에이전트(최기영) + 민주화(김재구) + 비전문가(김선경) + 노코드(이하석) + AI(전병기) + 지표(김대식)
