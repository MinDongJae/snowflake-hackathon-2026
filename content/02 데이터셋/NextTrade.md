# NextTrade (넥스트레이드)

> 한국 최초 대체거래소 — 2025년 3월 개장, 약 800개 종목

![[datasource_01_nexttrade.png]]

## 회사 배경
- 2022년 설립 (금융투자협회, 코스콤, 주요 증권사 34개사 공동 출자)
- KRX(한국거래소) 외에 주식을 거래할 수 있는 **두 번째 시장**
- 쉽게 말해: 학교 매점(KRX) 외에 편의점(NextTrade)이 하나 더 생긴 것

## 제공 데이터 (6종)

| 테이블 | 설명 |
|--------|------|
| 온라인시세B6우선호가 | 매수/매도 최대 10단계 호가 |
| 온라인시세E1종가매매호가 | 장 마감 고정가격 거래 |
| 온라인시세A3체결 | 실제 체결 가격/수량 |
| 온라인통계B5현정락 | 기준가 대비 가격 등락 |
| 온라인통계P0/C3프로그램매매 | 투자자별 알고리즘 거래 |
| 배치참조A0종목정보 | 종목 기본 정보 |

## 비즈니스 아이디어
- 🔥 NextTrade vs KRX 가격 차이 분석 → 차익거래 기회
- 호가 스프레드로 유동성 변화 추이 분석
- 프로그램매매 데이터로 기관/개인 투자 행태 비교

## 심사위원 관심 포인트
- **최기영** (SF 지사장): 마켓플레이스 데이터 활용
- **김재구** (리치고): 투자 의사결정 데이터
- **전병기** (LG U+): AI 기반 패턴 분석
