# 📊 데이터셋 총정리

> 4개 파트너가 Snowflake 마켓플레이스를 통해 실제 데이터를 제공

![[datasource_05_overview.png]]

## 파트너별 상세

- [NextTrade](/02-데이터셋/NextTrade) — 주식시장 (투자/금융)
- [Richgo](/02-데이터셋/Richgo) — 부동산/인구이동
- [SPH](/02-데이터셋/SPH) — 유동인구/소비/자산
- [아정당](/02-데이터셋/아정당) — 통신/이사/렌탈/생활서비스

## 크로스 활용 전략 (고득점 핵심!)

![[data_cross_strategy.png]]

| 조합 | 아이디어 |
|------|---------|
| **Richgo + SPH** | 아파트 시세 + 주민 소득/소비 = "이 동네 집값이 적정한가" |
| **아정당 + Richgo** | 이사 추정 + 인구이동 + 시세 = "왜 이 동네로 이사 가는가" |
| **SPH + NextTrade** | 지역 소비 트렌드 + 주가 = "소비 증가 → 주가 상승?" |
| **🏆 4개 전부** | **"라이프 체인지 분석"** — 이사 → 동네 특성 → 소비 변화 → 투자 |

> 여러 데이터를 조합할수록 높은 점수!
