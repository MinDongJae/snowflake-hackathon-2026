## 득점/감점 분석

### 1위 박준호 — 왜 1위였는가

| 항목 | 평가 | 분석 |
|------|------|------|
| **주제 독창성** | ⭐최고 | 대체 데이터 투자 = 참신한 발상 |
| **데이터 조합** | ⭐최고 | 백화점+카드+주가 = 의외의 크로스 |
| **타겟 시장** | ⭐최고 | 개인투자자 = 거대 시장 |
| **스토리** | ⭐최고 | "정보 비대칭 해소" = 감정적 공감 |

> 심사 코멘트: "정보 접근성이 낮은 개인투자자에게 객관적 데이터 지표를 제공하여 시장 평균을 상회하는 수익 창출 가능성을 제시"

### 2~4위 공통 약점

- 상권 분석, 아파트 예측 = **이미 많이 다뤄진 주제**
- 단일 도메인 내 데이터 조합 (의외성 부족)
- "분석 도구 개선" = 기능적 개선 (감정적 울림 약함)

## 순위를 가른 결정적 차이

| | 1위 | 2~4위 |
|---|-----|-------|
| **주제** | 참신 (대체 데이터) | 흔함 (상권/부동산) |
| **데이터** | 의외의 크로스 | 단일 도메인 |
| **타겟** | 거대 시장 | 좁은 시장 |
| **메시지** | 감정적 공감 | 기능적 개선 |

## 핵심 교훈

> **"어떤 문제를 푸느냐"가 "얼마나 잘 만드느냐"보다 더 중요하다**

1. "누구나 하는 주제"는 피하라
2. Marketplace 데이터를 "의외의 방식"으로 크로스
3. "감정적 공감"이 가능한 문제를 정의하라
4. 10분 영상: 문제 → 솔루션 → 임팩트
