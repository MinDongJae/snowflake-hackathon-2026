# ❄️ Snowflake란?

> 쉽게 말해: **인터넷에 있는 초대형 데이터 창고**

![[beginner_01_what.png]]

## 한 줄 설명

여기저기 흩어진 데이터를 **한 곳에 모아서**, **누구나 쉽게 분석**할 수 있게 해주는 클라우드 서비스

## 핵심 기능 4가지

![[beginner_02_features.png]]

| 기능 | 비유 | 설명 |
|------|------|------|
| **데이터 창고** | USB 대신 클라우드 | 무한 저장 가능한 온라인 저장소 |
| **마켓플레이스** | 앱스토어처럼 구매 | 다른 회사의 데이터를 클릭 한번으로 가져다 쓸 수 있는 장터 |
| **Cortex AI** | 말로 질문하면 답변 | AI가 자동으로 데이터를 분석해주는 기능 |
| **Streamlit** | 코딩 조금으로 웹앱 | 분석 결과를 예쁜 웹 앱으로 만들어주는 도구 |

## 기존 DB와 뭐가 다를까?

![[beginner_04_compare.png]]

| 기존 DB | Snowflake |
|---------|-----------|
| ❌ 서버 직접 관리 | ✅ 관리 필요 없음 |
| ❌ 용량 한계 있음 | ✅ 무한 확장 가능 |
| ❌ 설치 필요 | ✅ 웹 브라우저 접속 |
| ❌ 혼자만 사용 | ✅ 데이터 공유 쉬움 |

> **핵심 차이**: 인터넷만 있으면 OK. 관리 불필요.
